Getting Started

Follow these simple steps to set up the HTTPS server for your Godot project:

  1. Clone the repository
  2. Configure your SSL certificates
  3. Run the server
  4. Connect from your Godot project
# Start the server
cargo run --release

Configuration

Custom HTTPS certificate path configuration:

CERT_PATH=./cert.pem
KEY_PATH=./key.pem
